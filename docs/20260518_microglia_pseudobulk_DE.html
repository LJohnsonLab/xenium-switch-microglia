Microglia pseudobulk DE: control (E4) vs switched (E2)

On the microglia sub-cluster object at Louvain 0.5

Published

May 24, 2026

Differential expression on the microglia sub-cluster build (data/20260517_microglia_subcluster.qs2, Louvain 0.5, 11 sub-clusters) to compare the E4 control microglia (genotype label 4s2, condition = "control") against the E2 switched microglia (genotype label 4s2M, condition = "switched").

Pseudobulk per sample (7 sections: three 4s2 control sections, four 4s2M switched sections). Pseudobulk aggregation collapses each sub-cluster’s cells per sample into a single sample-level count vector before DESeq2. The unit of replication becomes the biological sample (n = 3 control, n = 4 switched).

Code 
# Seurat v5 holds the microglia sub-cluster object with the raw Xenium counts assay we aggregate from.
library(Seurat)
library(tidyverse)
# qs2 reads the multi-GB checkpoint roughly 10x faster than saveRDS.
library(qs2)
# presto gives a faster FindMarkers backend for the non-DESeq2 paths anayet can take.
# DESeq2 itself is invoked by FindMarkers(test.use = "DESeq2").
library(DESeq2)
# EnhancedVolcano renders the volcano plot inside anayet().
library(EnhancedVolcano)
# patchwork + gridExtra combine the volcano with the significant-DEG table side by side.
library(patchwork)
library(gridExtra)
library(kableExtra)

# Source the shared helper functions; anayet() wraps FindMarkers + EnhancedVolcano for the pseudobulk DE.
source("R/20260514-helper_functions.R")

# Seed any stochastic step so the results are reproducible across renders.
set.seed(1)

# Reload the microglia sub-cluster checkpoint built by 20260517_microglia_subcluster.qmd.
mg <- qs_read("data/20260517_microglia_subcluster.qs2")

# AggregateExpression pulls from the DefaultAssay's counts layer; we want the raw Xenium counts,
# not the SpaNorm log-residuals that are the analysis-default assay on this object.
DefaultAssay(mg) <- "Xenium"

Pseudobulk DE on the full microglia object

Merge every bona fide microglia sub-cluster (0, 2, 4, 6, 10) and the likely-contaminated clusters (1, 3, 5, 7, 8, 9) into a single sample-level pseudobulk contrast.

Positive log2FC means up in the E2-switched arm.

Code 
# Pool the five bona fide microglia clusters; drop the contamination clusters so the per-sample pseudobulks
# are not contaminated by neuronal / astrocyte / OL / vascular transcripts segmented into microglia masks.
# mg_pool <- subset(mg, clusters_res05 %in% c("0", "2", "4", "6", "10"))


# Uses the full microglia object
mg_pool <- mg

# Aggregate raw Xenium counts per sample; the resulting Seurat object has one column per biological sample.
bulk_pool <- AggregateExpression(mg_pool,
                                 assays        = "Xenium",
                                 group.by      = "sample_id",
                                 return.seurat = TRUE)

# AggregateExpression rewrites sample IDs ("4s2_F1" -> "g4s2-F1"); rebuild the lookup so condition attaches.
sample_meta_pool <- mg_pool@meta.data |>
  distinct(sample_id, condition) |>
  mutate(bulk_id = paste0("g", gsub("_", "-", sample_id)))

bulk_pool$condition <- sample_meta_pool$condition[match(colnames(bulk_pool), sample_meta_pool$bulk_id)]
stopifnot(!any(is.na(bulk_pool$condition)))
Idents(bulk_pool) <- "condition"

# Run DESeq2 through anayet() with the same thresholds as the per-state loop, for like-for-like comparison.
res_pool <- anayet(bulk_pool,
                   ident1          = "switched",
                   ident2          = "control",
                   test_method     = "DESeq2",
                   p_adj_threshold = 0.1,
                   min_pct         = 0.3,
                   plot_title      = "Pseudobulk DE (full microglia pool)",
                   print_plot      = FALSE)

# Always label the 5 lowest-p genes (blue) plus any FDR < 0.1 hits (red), mirroring the per-state plots so
# the volcano stays readable even when no gene clears FDR.
top5_p_pool    <- res_pool$all_results |> as_tibble(rownames = "gene") |> slice_min(p_val, n = 5, with_ties = FALSE) |> pull(gene)
sig_genes_pool <- rownames(res_pool$significant_results)
lab_genes_pool <- union(sig_genes_pool, top5_p_pool)

keyvals_pool <- rep("grey60", nrow(res_pool$all_results))
names(keyvals_pool) <- rownames(res_pool$all_results)
keyvals_pool[top5_p_pool]    <- "royalblue"
keyvals_pool[sig_genes_pool] <- "red2"

fc_range_pool <- range(res_pool$all_results$avg_log2FC, na.rm = TRUE)
y_max_pool    <- max(-log10(res_pool$all_results$p_val), na.rm = TRUE)

res_pool$volcano_plot <- EnhancedVolcano(res_pool$all_results,
                                         lab            = rownames(res_pool$all_results),
                                         selectLab      = lab_genes_pool,
                                         x              = "avg_log2FC",
                                         y              = "p_val",
                                         title          = "Pseudobulk DE (full microglia)",
                                         subtitle       = "switched (E2) vs. control (E4)",
                                         caption        = "BLUE: 5 lowest-p genes (FDR>0.1)\nRED: FDR < 0.1, min_pct = 0.3",
                                         labFace        = "italic",
                                         pCutoff        = 1,
                                         FCcutoff       = 0.001,
                                         colCustom      = keyvals_pool,
                                         legendPosition = "none",
                                         drawConnectors = TRUE,
                                         directionConnectors = "both",
                                         max.overlaps   = Inf,
                                         xlim           = c(floor(fc_range_pool[1]), ceiling(fc_range_pool[2])),
                                         ylim           = c(0, max(ceiling(y_max_pool * 1.05), 1)))

# Fall back to the 5 lowest-p genes when no gene clears FDR < 0.1, so the side table is never empty.
if (nrow(res_pool$significant_results) > 0) {
  table_pool <- res_pool$significant_results |>
    as_tibble(rownames = "gene") |>
    select(gene, p_val, avg_log2FC, p_val_adj) |>
    mutate(across(where(is.numeric), \(x) signif(x, 3))) |>
    arrange(p_val_adj)
} else {
  table_pool <- res_pool$all_results |>
    as_tibble(rownames = "gene") |>
    slice_min(p_val, n = 5, with_ties = FALSE) |>
    select(gene, p_val, avg_log2FC, p_val_adj) |>
    mutate(across(where(is.numeric), \(x) signif(x, 3)))
}
table_grob_pool <- tableGrob(table_pool, theme = ttheme_minimal(base_size = 12), rows = NULL)

# 3:4 width ratio matches the per-state combined panels.
res_pool$volcano_plot + wrap_elements(table_grob_pool) +
  plot_layout(widths = c(3, 4))

de_results <- imap(microglia_states, \(cluster_ids, label) {
  # Subset to the cells of one microglia state; %in% accepts both single ids and bound id-vectors.
  sub <- subset(mg, clusters_res05 %in% cluster_ids)

  # Aggregate raw Xenium counts per sample; return.seurat = TRUE produces a sample-level Seurat object.
  bulk <- AggregateExpression(sub,
                              assays        = "Xenium",
                              group.by      = "sample_id",
                              return.seurat = TRUE)

  # AggregateExpression silently replaces underscores in sample IDs with hyphens, and prepends "g" to any
  # group label that starts with a digit ("starts with a number, appending `g` to ensure valid variable names").
  # Our sample IDs (4s2_F1, 4s2M_F1, ...) hit both rules, so we mirror them when matching the per-sample
  # condition labels back to the pseudobulk column names.
  sample_meta <- sub@meta.data |>
    distinct(sample_id, condition) |>
    mutate(bulk_id = paste0("g", gsub("_", "-", sample_id)))

  bulk$condition <- sample_meta$condition[match(colnames(bulk), sample_meta$bulk_id)]
  # Hard-stop if the metadata join produced NA; common cause is forgetting the underscore -> hyphen fix.
  stopifnot(!any(is.na(bulk$condition)))
  Idents(bulk) <- "condition"

  # Run the DE through anayet()
  # plot_title is left short because anayet() appends " in <cell_type>" derived from ident1; we rebuild the volcano below.
  res <- anayet(bulk,
                ident1          = "switched",
                ident2          = "control",
                test_method     = "DESeq2",
                p_adj_threshold = 0.1,
                min_pct         = 0.3,
                plot_title      = "Pseudobulk DE",
                print_plot      = FALSE)

  # Rebuild the volcano so the 5 genes with the lowest raw p-value are always labelled, even when the
  # FDR-significant set is empty (most p_val_adj sit at 1 in low-power pseudobulks on a 480-gene panel).
  # Significant-at-FDR-0.1 genes stay in red; the top-5-by-p-value sit in blue if they are not already significant.
  top5_p   <- res$all_results |> as_tibble(rownames = "gene") |> slice_min(p_val, n = 5, with_ties = FALSE) |> pull(gene)
  sig_genes <- rownames(res$significant_results)
  lab_genes <- union(sig_genes, top5_p)

  keyvals <- rep("grey60", nrow(res$all_results))
  names(keyvals) <- rownames(res$all_results)
  keyvals[top5_p]   <- "royalblue"
  keyvals[sig_genes] <- "red2"

  fc_range  <- range(res$all_results$avg_log2FC, na.rm = TRUE)
  y_max     <- max(-log10(res$all_results$p_val), na.rm = TRUE)

  res$volcano_plot <- EnhancedVolcano(res$all_results,
                                      lab            = rownames(res$all_results),
                                      selectLab      = lab_genes,
                                      x              = "avg_log2FC",
                                      y              = "p_val",
                                      title          = paste("Pseudobulk DE in", label),
                                      subtitle       = "switched (E2) vs. control (E4)",
                                      caption        = "BLUE: 5 lowest-p genes (FDR>0.1)\nRED: FDR < 0.1, min_pct = 0.3",
                                      labFace         = "italic",
                                      pCutoff        = 1,
                                      FCcutoff       = 0.001,
                                      colCustom      = keyvals,
                                      legendPosition = "none",
                                      drawConnectors = TRUE,
                                      directionConnectors = "both",
                                      max.overlaps   = Inf,
                                      xlim           = c(floor(fc_range[1]), ceiling(fc_range[2])),
                                      ylim           = c(0, max(ceiling(y_max * 1.05), 1)))

  # Emit a QMD section header so the rendered report is navigable.
  cat("\n\n### ", label, "\n\n", sep = "")
  cat("Pseudobulk DESeq2 on", ncol(bulk), "samples (", sum(bulk$condition == "control"),
      "control vs", sum(bulk$condition == "switched"), "switched).",
      "Total genes tested:", res$summary$total_genes,
      "| Significant at FDR < 0.05:", res$summary$significant_genes, "\n\n")

  # Build the table grob. When FDR-significant hits exist, show them. Otherwise fall back to the 5 genes
  # with the lowest raw p-value so the table mirrors the volcano labels and the user can still read the
  # top of the ranked list even when no gene clears FDR < 0.1.
  if (nrow(res$significant_results) > 0) {
    sig <- res$significant_results |>
      as_tibble(rownames = "gene") |>
      select(gene, p_val, avg_log2FC, p_val_adj) |>
      mutate(across(where(is.numeric), \(x) signif(x, 3))) |>
      arrange(p_val_adj)
    table_grob <- tableGrob(sig, theme = ttheme_minimal(base_size = 12), rows = NULL)
  } else {
    top5 <- res$all_results |>
      as_tibble(rownames = "gene") |>
      slice_min(p_val, n = 5, with_ties = FALSE) |>
      select(gene, p_val, avg_log2FC, p_val_adj) |>
      mutate(across(where(is.numeric), \(x) signif(x, 3)))
    table_grob <- tableGrob(top5, theme = ttheme_minimal(base_size = 12), rows = NULL)
  }

  # Combine the volcano with the significant-DEG table side by side; 2:5 width ratio mirrors the skill default
  # adjusted for the wider table we want here.
  combined <- res$volcano_plot + wrap_elements(table_grob) +
    plot_layout(widths = c(3, 4))
  print(combined)

  # Emit the per-cluster expected-biology vs contamination commentary below the panel; results: asis renders
  # the cat'd markdown verbatim.
  cat("\n\n", cluster_commentary[[cluster_ids]], "\n\n", sep = "")

  res
})

# Persist the full DESeq2 result tables (all genes, not just significant) per microglia state.
# Filename slug pulls the cluster id(s) from the label (e.g. "clusters 2 + 6" -> "2_6", "cluster 0" -> "0").
iwalk(de_results, \(res, label) {
  slug <- label |>
    str_extract("clusters? [\\d \\+]+") |>
    str_remove("clusters? ") |>
    str_replace_all(" \\+ ", "_") |>
    str_trim()
  res$all_results |>
    as_tibble(rownames = "gene") |>
    write_csv(paste0("results/20260518-pseudobulk_DE_microglia_cluster", slug, "_switched_vs_control.csv"))
})

Likely segmentation contamination in non-APOE hits. Several top-5 lowest-p genes across states are not microglial: Cnr1 and Slc17a7 (excitatory neurons) in cluster 4; Pdgfra (OPC) in cluster 2; Ms4a4b (T cells) in cluster 6; Plcb4 and Tunar (neurons) in the full microglia pool. Xenium boundary-stain segmentation bleeds transcripts across adjacent cells, especially at boundaries with high transcript density, and DAM microglia near amyloid plaques carry more high-transcript neuronal neighbors than homeostatic microglia. The cluster-6 boundary leaks (Ttr / Ermn / Apod in its top-10 per the assignment note) already flag this mechanism. A per-cell AddModuleScore() filter against the seven contamination-marker sets is the proper next step before treating any of these non-microglial hits as biology.

Caveats

  • Cluster 10 (IRM + proliferating) is sparse on three Slide1 samples. 4s2_F1 contributes 11 cells, 4s2M_F1 contributes 23 cells, and 4s2M_F2 contributes 23 cells; the per-sample pseudobulk counts in those three replicates will be low and DESeq2 dispersion estimates may be unstable. 4s2M_F3 carries 139 of 399 cluster-10 cells (35 %), so a single sample drives a disproportionate fraction of the switched-arm signal.
  • lack of statistical power. Pseudobulk DE on a 480-gene panel with n = 3 vs 4 samples is underpowered. FDR adjustment over 480 tests is stringent and the seven samples are at the lower limit DESeq2 can fit dispersion against. Only cluster 10 (with the strongest IRM / proliferation contrast) produces FDR-significant hits, and three of its four DEGs survive Benjamini-Hochberg only because their p-values are extreme (≤ 2e-5).
