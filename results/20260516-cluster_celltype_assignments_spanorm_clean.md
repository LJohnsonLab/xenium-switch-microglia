# Cluster cell-type assignments — SpaNorm clean (cluster 20 removed), sample.p = 0.25, resolution 0.5

I assigned each of the 22 Louvain clusters from `20260516_SpaNorm_samplep025_clean.qmd` using the top 10 FindAllMarkers per cluster (ranked by `avg_log2FC`; source: `results/20260516-find_all_markers_spanorm_p025_clean.csv`). The panel is the 480-gene Xenium mBrain panel, so the cell-type vocabulary is constrained to what those probes resolve. I ordered the rows by cell-type similarity, not by cluster number, so related populations sit next to each other. The "Flagged genes" column lists top-10 markers that do not fit the called identity and that I would scrutinize as possible doublet or segmentation contamination before downstream DE.

**Scope note.** The build chunk filters `seurat_clusters %in% c("20", "25")` from `data/20260512_microglia_switch_spanorm_p025.qs2`. That object only carries cluster levels 0-24, so the `"25"` term matched no cells. Only the cluster-20 cells (Calb1+/Tdo2+ excitatory neurons of uncertain region from the parent annotation) were dropped before the per-sample SpaNorm re-fit. The cluster count nonetheless drops from 25 to 22 because Louvain at resolution 0.5 collapses two additional clusters once the cluster-20 cells are removed and the PCA is re-computed on the smaller cell set.

## Summary by lineage

I split the 22 clean clusters into:

* **Oligodendrocyte lineage (3 clusters):** OPCs (13) and two mature OL clusters (0, 9). Identical structure to the parent build. The cluster-20 removal did not perturb the OL lineage at all.
* **Astrocytes (2 clusters):** gray-matter Aqp4+ (4) and Agt+ subtype (8). Cluster 4 now carries a low-pct.1 Cxcl10 signal that was absent from the parent equivalent, hinting at IFN-responsive astrocytes inside the gray-matter cluster.
* **Microglia (1 cluster):** cluster 2, still mixing homeostatic and DAM markers, with DAM markers Cst7 and Cd14 newly in top-10. Sub-cluster before DE.
* **T cells (0 clusters, REGRESSION):** the discrete cytotoxic-T-cell cluster from the parent (parent 24: Cd3d / Gzmb / Nkg7 / Itk) does not appear in the clean build. T-cell markers redistribute to cluster 7 (VLMC / perivascular) at low pct.1 (Cd3d at 0.022, Itk at 0.031). Removing cluster-20 cells and re-clustering at the same Louvain resolution lost the T-cell signal.
* **Ependymal + choroid plexus (2 clusters):** 18 and 19. CP now carries MHC-II (H2-Eb1 / H2-Ab1), a new and plausible call. Ependymal cluster 18 still carries some Slide2 enrichment (ventricle sampling).
* **Vascular (3 clusters):** endothelial (5), pericyte (17), VLMC (7). Endothelial top-10 is cleaner than the parent; VLMC absorbs the missing T cells.
* **Excitatory neurons (6 clusters):** deep-layer Fezf2+ (1), upper-layer Cux2+/Lamp5+ (6, now blended with the parent's inhibitory Lamp5+/Lypd1+), Vglut2+/Rorb+ (11, now also Calb2+), Calb2+/Tac1+/Nts+ mixed (3), Rspo1+/Tcap+/IEG-rich new cluster (15), and the low-specificity IEG cluster (20).
* **GABAergic interneurons (3 clusters, COLLAPSED):** mixed Vip+/Sst+/Lhx6+/Pvalb+ (10), Pvalb+/Trh+ (21), and striatal-flavored Tac1+/Adcy5+/Pax6+ (12). The parent's four-way GABAergic split (Sst 11, Pvalb 22, Vip 21, Lamp5/Lypd1 14) collapses to three, with the Sst / Vip / Lhx6 separation lost inside cluster 10.
* **Mixed / disease-associated (2 clusters):** 14 (stressed neurons) and 16 (reactive + Tdo2). Cluster 16 also absorbs the Tdo2+ residue of parent cluster 20 that was not dropped (only cluster-20-labelled cells were removed).

The headline findings:

1. **T-cell cluster lost.** The parent's most consequential SpaNorm-vs-SCT win was the discrete cytotoxic-T-cell cluster (parent 24). The clean build does not recover it at resolution 0.5; the lymphocyte markers scatter into the VLMC / perivascular cluster (clean 7) at pct.1 ≤ 0.03. Removing cluster-20 cells does not biologically explain this loss (cluster 20 was an excitatory population), so the most likely mechanism is that the smaller cell budget plus the re-computed PCA shifts the local density and Louvain merges the small T-cell island into its perivascular neighbor. **To recover the T cells, re-run FindClusters at a higher resolution (e.g. 0.7-0.9) on the clean object before doing any DE.**
2. **GABAergic resolution drops.** The parent separated Sst+, Vip+, and Lamp5+/Lypd1+ inhibitory subclasses. The clean build collapses Sst + Vip + Lhx6 into one mixed cluster (clean 10). Same fix: higher Louvain resolution on the clean object.
3. **One new excitatory cluster (clean 15, Rspo1+/Tcap+/IEG-rich)** appears that does not directly match any parent cluster. It looks like a partial recombination of parent cluster 19 (Tshz2+) with the non-cluster-20 Tdo2+ neighbors, in a high-IEG state. Spatial inspection required before interpretation.
4. **Faint signals gained.** The astrocyte cluster now carries Cxcl10 (IFN-responsive astrocyte hint), the microglia cluster now shows DAM markers (Cst7, Cd14) in top-10, the ependymal cluster carries C3 / Lgals3 (reactive), and the choroid plexus carries MHC-II (H2-Eb1 / H2-Ab1). None of these were in the parent top-10s. Removing cluster-20 cells and re-running the per-sample SpaNorm appears to sharpen subtle reactive / immune signals in the cells that remain. This is biologically interesting and consistent with reducing one large excitatory cluster's pull on the shared variance.

## Verdict and recommended cleanup before downstream DE

Removing cluster 20 (Calb1+/Tdo2+ excitatory of uncertain region) was justified, but the re-fit at the same Louvain resolution **loses the T-cell cluster and collapses three GABAergic subclasses into one**. The clean build is a usable scaffold for the gray-matter, white-matter, and vascular comparisons, but I would not promote it over the parent for the immune / lymphocyte questions without recovering the T cells first.

I would (1) re-run `FindClusters(resolution = 0.7)` or `0.9` on the clean object to recover the parent's T-cell cluster and the parent's three-way GABAergic split, (2) confirm cluster 15 (Rspo1+/Tcap+/IEG-rich) is not a dissociation artefact by inspecting its spatial location, (3) check whether Cxcl10 inside cluster 4 corresponds to a localized IFN-responsive astrocyte population by sub-clustering astrocytes (4, 8), (4) sub-cluster microglia (2) to separate homeostatic from DAM cleanly given that DAM markers now sit in top-10, (5) run DoubletFinder on clusters 14 and 16, (6) merge or re-resolve cluster 20 (low-specificity IEG), and (7) flag the "cluster 25" the user originally asked to drop as a non-existent cluster (the parent build only has 0-24); only cluster-20 cells were removed.

Once the T-cell cluster is recovered, the clean build is a strict improvement over the parent for everything except the Vip / Sst / Lhx6 inhibitory separation, which needs the higher Louvain resolution either way.
