# results/

CSV outputs from the analysis notebooks. Two families of files live here:

1. **`*-find_all_markers_*.csv`** — presto-backed `FindAllMarkers(only.pos = TRUE)` tables. One row per (cluster, gene); columns are `p_val`, `avg_log2FC`, `pct.1`, `pct.2`, `p_val_adj`, `cluster`, `gene`. Each file is generated by a single qmd and consumed by the same qmd (top-10-per-cluster table, `fiddle()` violin panel) and by the matching cell-type assignment note under `notes/`.
2. **`*-pseudobulk_DE_*.csv`** — DESeq2 result tables from the pseudobulk switched-vs-control contrast on the microglia sub-cluster (`anayet()` + `DESeq2::results()`, all genes, unshrunken `log2FoldChange`).

Date prefix in every filename is the run date the file was first generated (`YYYYMMDD-`).

## How to regenerate

`FindAllMarkers` and `DESeq2` chunks are wrapped in `#| eval: false` after their first render so reload is fast. To regenerate a file, set `eval: true` on the corresponding chunk and re-render the qmd; the chunk reads the upstream `.qs2` checkpoint from `data/` and writes the CSV here.
